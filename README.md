# React Quiz Application

A simple Quiz Application built using **React.js**.  
This project displays questions one by one and calculates the user's score based on correct answers.

## Features
- Displays quiz questions
- Accepts user answers
- Calculates score
- Shows final result

## Technologies Used
- React.js
- JavaScript
- HTML
- CSS

## Project Structure
src/
 ├── App.jsx
 ├── main.jsx

## How it Works
1. The application displays a question.
2. The user enters an answer using a prompt.
3. If the answer is correct, the score increases.
4. After all questions are answered, the final score is displayed.
